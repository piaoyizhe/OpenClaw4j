# 学校招标网站信息获取工作手册

## Description
提供获取和分析学校招标网站信息的标准化流程，包括招标信息的抓取、解析、筛选和报告生成等任务。优先访问宁夏公共资源中心和宁夏工商职业技术大学的招标网站信息。

## Input
用户的招标信息获取请求，包含以下信息：
- 学校名称：目标学校的名称
- 招标网站URL：学校招标网站的地址（可选）
- 关键词：搜索的关键词（可选）
- 时间范围：查询的时间范围（可选）
- 信息类型：招标公告、中标公告、更正公告等（可选）

## Output
- 执行结果：操作是否成功完成
- 招标信息列表：获取到的招标信息，包含标题、发布日期、链接、招标名称、金额等
- 详细报告：执行过程和结果的详细信息
- 错误信息：如果操作失败，提供错误原因

## Steps
1. **分析用户请求**：理解用户想要获取哪个学校的什么类型的招标信息
2. **验证输入参数**：检查学校名称是否正确
3. **制定执行计划**：根据学校名称选择优先访问的网站
4. **执行网络操作**：
   - 优先访问宁夏公共资源中心网站 `https://ggzyjy.fzggw.nx.gov.cn/`
   - 对于宁夏工商职业技术大学，特别关注 `https://www.nxgs.edu.cn/xxgk1/cggk1.htm`
   - 其他学校可以使用百度搜索查找招标网站
5. **解析页面内容**：提取招标信息列表
6. **访问详情页**：逐个访问招标信息详情页，读取招标名称、金额等详细信息
7. **筛选和整理**：根据关键词和时间范围筛选信息，整理成结构化数据
8. **生成执行报告**：向用户提供详细的招标信息报告

## Examples
### 示例1：获取宁夏工商职业技术大学招标信息
**输入**：获取宁夏工商职业技术大学的最新招标公告
**步骤**：
1. 访问宁夏工商职业技术大学招标网站 `https://www.nxgs.edu.cn/xxgk1/cggk1.htm`
2. 解析页面，提取招标公告列表
3. 逐个访问详情页，读取招标名称、金额等信息
4. 整理招标信息，按发布日期排序
5. 生成招标信息报告

### 示例2：获取宁夏地区学校招标信息
**输入**：获取宁夏地区学校的计算机设备招标信息
**步骤**：
1. 访问宁夏公共资源中心网站 `https://ggzyjy.fzggw.nx.gov.cn/`
2. 使用关键词"计算机设备"和"学校"进行搜索
3. 解析搜索结果，提取招标信息列表
4. 逐个访问详情页，读取招标名称、金额等信息
5. 整理筛选后的信息
6. 生成招标信息报告

### 示例3：获取其他地区学校招标信息
**输入**：获取北京大学的最新招标公告
**步骤**：
1. 使用百度搜索"北京大学 招标网站"
2. 解析搜索结果，找到北京大学招标网站
3. 访问招标网站，获取页面内容
4. 解析页面，提取招标公告列表
5. 逐个访问详情页，读取招标名称、金额等信息
6. 整理招标信息，按发布日期排序
7. 生成招标信息报告

## Notes
- **优先访问**：优先访问宁夏公共资源中心和宁夏工商职业技术大学的招标网站
- **网站结构**：不同学校的招标网站结构可能不同，需要根据实际情况调整解析策略
- **反爬措施**：注意网站的反爬措施，避免频繁请求导致IP被封
- **详情页访问**：必须访问详情页获取完整的招标信息，包括招标名称、金额等
- **数据解析**：确保正确解析页面内容，提取完整的招标信息
- **错误处理**：妥善处理网络错误和页面解析错误
- **信息准确性**：验证获取到的信息的准确性和完整性
