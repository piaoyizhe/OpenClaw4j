# 版本更新日志

## 下一步开发计划

### 高优先级功能
- **向量语义搜索** - 替代关键词搜索，提升记忆检索准确率
- **定时任务系统** - 支持定时执行工作流、发送日报等自动化任务
- **语音交互** - 语音输入和播报功能
- **Web UI 界面** - 更友好的用户界面

### 中优先级功能
- **API 服务化** - 封装为 REST API，支持Webhook
- **代码执行能力** - 安全执行Python/Shell代码
- **多Agent协作** - 多个专业Agent分工协作
- **知识图谱** - 记忆内容结构化，实体关系提取

## 历史版本

- 核心功能：大模型集成、记忆系统、工作流管理
